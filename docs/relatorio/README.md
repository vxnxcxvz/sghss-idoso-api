# Relatório

Este diretório deve conter o relatório final em PDF do trabalho, conforme exigido pela disciplina.
Nomeie o arquivo de maneira descritiva (por exemplo, `SGHSS_project_report.pdf`) e armazene-o aqui.
Utilize este espaço também para outros arquivos de referência que acompanharem o relatório (por exemplo, scripts SQL ou documentação complementar).

Se o relatório final ainda não estiver pronto, mantenha marcado como [PENDENTE] no documento principal até que você faça o upload do PDF neste diretório.
