# Diagramas do SGHSS (modo seguro)

## 1) DER (Entidade-Relacionamento)
- Site: https://dbdiagram.io/
- Cole `DER-code.dbml`
- Exporte PNG e salve como `docs/diagramas/der.png`

## 2) Casos de Uso (UML)
- Site: https://app.diagrams.net/
- Atores mínimos: Paciente, Profissional, Administrador
- Casos mínimos (escopo entregue): register/login/me + listar/criar pacientes
- Salvar como `docs/diagramas/casos-de-uso.png`

## 3) Diagrama de Classes (UML)
- Site: https://app.diagrams.net/
- Classes mínimas: Usuario, Paciente (+ relações básicas)
- Salvar como `docs/diagramas/diagrama-classes.png`
